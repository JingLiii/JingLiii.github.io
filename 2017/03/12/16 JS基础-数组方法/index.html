JS基础-数组常用方法

Array

数组为JS提供的内置对象

属性

  • .length属性
  • 数组中最大索引值 + 1 ;

方法

  • instanceof

    • 判断某个对象,是否属于某个构造函数
    • 前面是需要判断的对象, 后面是构造函数
    • 此为表达式,不需要进行调用使用

  • Array.isArray(需要判断的对象)

    • 返回值为Boolean类型

  • toString()

    • 将数组转化为字符串
    • 每一项用 , 进行分割

  • valueOf()

    • 返回数组对象本身.(‘’’’’’真不知道有什么用’’’’’’)

  • join(分隔符)

    • 返回一个字符串
    • 使用传入的分隔符,将各项链接起来
    • 不传参,默认为 ,

  • 方法 : push pop unshift shift

  • reverse()

    • 翻转数组
    • 将原数组进行翻转
    • 并将翻转后的数组返回

  • sort(fn)

    • 需要传入回调函数
    • 回调函数的两个参数,代表前后的两个值
    • 排序规则根据比较两个参数后, 返回 的 大于0,等于0,小于0 进行排序

  • concat(需要拼接的数据)

    • 将参数拼接到当前的数组中
    • 如果传入的是数组,会拆分每一项数据进行拼接
    • 如果是对象,会将整个对象放到最后一项

  • slice(开始的索引,结束的索引)

    • 从开始索引开始截取,包括开始索引
    • 截取到结束索引, 不包括结束索引
    • 返回值是截取到的新数组

  • splice(开始索引,结束索引,替换的数据)

    • 从开始位置开始截取,包括开始位置
    • 截取到结束索引,不包括结束索引
    • 截取的数据作为新数组返回
    • 原数组失去被截取的数据,并使用替换位置数据进行替换

  • indexOf(需要查找的数据),

  • lastIndexOf():从后往前查找

    • 返回元素所在的索引
    • 若是不存在返回 -1

  • every(fn)

    • 参数需要传入回调函数
    • 函数参数代表每一个元素
    • 返回一个判断条件,若是都满足的话 返回true, 有一项不满足 停止遍历 返回false

  • filter(fn)

    • 判断每一项是否满足条件,
    • 将满足条件,返回值为true的元素,组成新数组进行返回
    • 第三个参数表示数组本身

  • foreach(fn)

    • 遍历数组
    • 可以查看数据中的每一项
    • 但是没有返回值,也不能对原数组进行操作
    • 第三个参数表示数组本身

  • map(fn)

    • 对数组中的每一项进行操作
    • 将操作后的元素,形成一个新的数组返回
    • 第三个参数表示数组本身

  • some(fn)

    • 根据回调函数对每一个元素进行操作
    • 如果每一个回调函数的返回值都是false 就返回false
    • 有一个返回true的 就返回true
    • 第三个参数表示数组本身

  • 清空数组

    • array.splice(0,array.length); //删除数组中所有项目
    • array.length = 0; //length属性可以赋值，其它语言中length是只读
    • array = []; //推荐
