JS基础-对象基础

对象的概念

无序的属性集合:

基本概念

具有特指的一类叫做对象

  • 由属性和方法组成
  • 对象内部的变量叫做属性. 外部的变量叫做变量.
  • 对象内部的函数叫做方法. 外部的函数叫做函数

    绑定属性和方法

  • 方法一: 直接通过 . 进行绑定
  • 方法二: [变量名] 进行绑定
    • 此方法更为灵活

对象的创建

new Object();

new 后面调用的函数,我们称之为构造函数.Obeject()我们把他视之为构造函数,构造函数本质就是一个函数,只不过构造函数的目的是为了创建新对象,为新对象设置初始化(设置对象的属性)

  • new之后的过程
  1. 创建一个新对象
  2. 将构造函数的作用域赋值给新对象,所以this = person
  3. 执行构造函数,为person设置属性
  4. 返回新对象

    字面量

    直接通过 {} 的方式进行创建
