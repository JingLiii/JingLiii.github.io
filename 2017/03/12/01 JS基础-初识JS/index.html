JS基础-初识JS

JavaScript 的作用

js的最初目的

  • 判断客户端的输入

    现在的意义

  • 网页特效(PC端的网页效果)
  • 移动端(移动WEB和APP)
  • 异步与服务器进行交互
  • 服务器端开发(node.js)

    浏览器是如何运行的?

    关于javascript语言

  • 用的最多的脚本语言
    • 脚本语言:不需要编译,直接运行,边解析边运行
  • 客户端的脚本语言

    JavaScript组成

  • ECMAScript:JavaScript的语法规范
  • DOM: JavaScript操作页面上的元素的API
  • BOM: JavaScript操作浏览器部分功能API

    输出语句

  • console.log(“内容”): 在控制台打印输出内容
  • document.write(“内容”): 在页面书写内容
  • alert(“内容”): 弹出显示内容

    注释

  • 单行注释: //
  • 多行注释: / /
  • 注释的作用:
    • 一般用于解释某些复杂代码的逻辑,方便后期的维护与开发
    • 对整个代码块进行分割
      *　一般用于：模块，函数，复杂逻辑注解，文件注解，维护记录

      书写位置

  • 在HTML中直接书写,一般写在上面
  • 引入外部文件: <script src="地址"></script>
