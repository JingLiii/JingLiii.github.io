作用域 与 变量提升

变量的作用域

根据变量的作用范围不同,可以对变量进行分类

  • 全局变量
    • 定义在script 或者 不属于某个函数的变量
    • 在任何位置中,不带var的变量也是全局变量
  • 局部变量
    • 在函数内部带有var定义的变量
  • 作用域链
    • 函数内部可以访问到该函数所属的外部作用域的变量
    • 局部变量退出作用域之后就会销毁, 全局变量关闭网页或浏览器才会销毁

      预解析 与 变量提升

  • 预解析
    • 在执行代码前,先检查语法错误,再进行变量提升
  • 变量提升 ( 函数范围内同样有效 )
    • 预解析时,js把所有的 变量名 和 函数 全部提升到js的最顶端
    1. 只提升变量名, 不提升变量值. 函数整体进行提升
    2. 提升时 先提升变量,再提升函数
      使用变量,在变量定义之前,容易出现undefined
  • 就近原则: 局部变量和全局变量一样时,使用局部变量

  • 形参也是一种局部变量.(局部变量,函数执行完后,立即收回)

高级函数

函数的数据类型

函数是复杂数据类型,属于obejct的一种
较为特殊,打印时单独显示

  • 任何数据类型 都是字符串定义的.

函数的三个高级操作

函数做参数(回调函数).
函数自调用(递归).
函数做返回值(闭包).

  1. 回调函数

    • 函数作为参数
    • 参数传递时,传递的是函数名或者是函数体,若是直接放入执行函数,那么放入的就是返回值
    • 一旦参数名后加( ),就代表,这个参数一定是一个函数.
    • 那么这个函数就成为回调函数
    • 回调函数,就是我们定义一种规则,而具体的执行,和传入和参数,并不是我们控制的

  2. 递归

    • 函数自己调用自己
    • 递归必须有跳出条件,不然就是死循环,没有了意义
  3. 闭包
    • 函数的返回值,这是一个函数
    • 形成函数的嵌套
