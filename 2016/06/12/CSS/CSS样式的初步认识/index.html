CSS样式的初步认识

2016-06-12

CSS介绍与规则

CSS: Cascading Style Sheets 层叠样式表

规则:

  • 属性与属性值之间以”键值对”的形式出现.
  • 属性是对指定的对象设置的样式属性

    字体样式属性

    font-size: 子号大小

    font-family: 字体

  • 指定多个字体时,可以使用逗号进行隔开,浏览器依次寻找,找到第一个合适的字体为准.
  • 网页中普遍使用14px
  • 中文字体,字体中带有空格,#,$符号时,使用引号括起来.

    font-Wight: 字体粗细

  • 属性值包括: normal,bold,bolder,lighter,100-900
  • 400-normal,700-bold,900-bolder

    font-style: 字体风格

  • normal: 默认没有风格
  • italic: 显示斜体的字体样式
  • obique: 倾斜字体.

    字体的综合设置

    {font: font-style font-weight font-size/line-height font-family;}
  • 各个属性之间的位置不能发生调换.

    CSS注释

    / 需要注释的内容 /

    CSS外观属性

    color: 文本颜色

  • 三种定义的方式:文字定义,十六进制,RGBA.

    line-height: 行高

  • 行于行之间的高度,当行高等于某元素的高度时,其中的元素居中

    text-align: 水平对齐方式

  • left,right,center

    text-indent: 首行缩进
