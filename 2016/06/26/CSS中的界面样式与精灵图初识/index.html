CSS中的界面样式与精灵图初识

2016-06-26

CSS用户界面样式

cursor 鼠标样式

  • default: 小白
  • pointer: 小手
  • move: 移动
  • text: 文本
  • 尽量不要使用hand,因为火狐并不支持.

    轮廓

    绘制于元素周围的一条线,位于边框的外围,起到突出元素的效果

  • outline: outline-color||outline-style||outline-width
  • 使用 czoutline: 0 直接去除边框.

    防止文本拖拽

    resize: none这个单词可以防止火狐,谷歌浏览器拖拽文本框

    vertical-algin: 垂直对齐

    不影响块级元素中的内容对齐,只针对行内元素,或者行内块元素,特别是寒内块元素,用来控制图片和表单

    图片和文字对齐

  • 默认的是文字和图片的基线对齐
  • vertical: baseline 基线对齐(默认)
  • vertical: middle 文字与图片的中间对齐
  • vartical: top 文字与图片的顶部对齐

    去除图片底侧空白缝隙

    因为图片的底部默认对齐父级元素的基线

  • 解决方法
    • img vartical: middle|top 让图片不和基线对齐
    • img 添加display: block 图片转化为块级元素即可解决这个问题

      精灵图片技术

      很多的小图片放到一张背景图中,通过调整背景图片的位置,来达到我们想要展现的效果.
