CSS中的选择器, 元素分类以及CSS的引入规则

2016-06-16

符合选择器

交集选择器

  • 由标签选择器与类样式两个选择器构成,中间不能有空格
  • 同时符合两个条件的标签被选中

    并集选择器

  • 由逗号链接,可以同时对选中的多个样式,便签进行统一处理
  • 通常用于集体声明

    后代选择器

  • 使用空格进行隔开
  • 无论是子代,还是孙子辈的是,都可以被选中.只要是包含在内的标签.

    子元素选择器

  • 使用>号进行隔开,并在符号两边加空格
  • 只能选选中自代,无法选中其他的元素.

    显示模式

  • 只有文字才能组成段落,所有p标签,h标签中无法再放置其他的标签
  • 链接中不能再放链接

    块级元素

  • 独占一行
  • 能够设置宽高
  • block
  • 高度,行高,内外边距都能控制
  • 默认为容器的100%.
  • 可以容纳内联元素和其他块级元素

    • 行内元素

    • 不占有独立一行
    • 需要内容进行撑开,没有宽高
    • 只能解说文本和其他行内元素,但是特殊.
    • 只能设置水平方向的内外边距,垂直方向无效

      行内块元素

    • 和相邻的行内元素在一行上面,但是之间会有空隙.
    • 默认宽度是其本身的母暗度
    • 宽,高,内外边距都可以控制.

      显示模式转化

      通过display进行转换

    • block: 转化为块元素
    • inline: 转化为行内元素
    • inline-block: 转换为行内块元素

      CSS伪类,伪元素

      伪类

      用于文档状态的改变.动态事件

      text-decoration

    • none: 无下划线
    • underline: 下划线(默认值)
    • line-through: 删除线
    • overline: 上划线

      伪元素

      在C3中详细讲解.

      行高

    • 一段文字分为:顶线,中线,基线,底线
    • 行高等于与元素的高度,上距离与下距离相等,元素中的文本内容就会在元素中保持垂直集中.
    • 行高设置单位直接计算,没有设置单位格局上一层的字体大小进行计算.

      引入CSS样式表

      行内样式-内联样式

      通过元素的style属性进行设置

    内部样式表-内嵌式

    CSS代码集中写在head标签中

    外部样式表-外链式 (推荐使用)

    将所有的样式放到一个css文件中,通过link引入到html文件中

    • herf: 外部样式文件的地址
    • type: 定义所连接文档的类型
    • rel: 规定当前文档与被连接文档之间的关系

      导入式

      在style标签的开头使用@import语句,即可导入外部样式表文件

    CSS三大特性

    层叠性

    权重相同的情况下,后面的样式,会覆盖前面的样式

    继承性

    父元素含有某属性,子元素也会应用这个属性
    一般情况下:font-,text-,line-,color-,开头的属性,可以被继承

    优先级-权重问题

    • 使用了!important声明的规则
    • 内嵌在html元素的style属性
    • 使用了id选择器的规则
    • 使用了类选择器,属性选择器(在C3中使用中括号),伪元素和伪类选择器的规则
    • 使用的元素标签选择器
    • 只包含一个通用选择器规则.
