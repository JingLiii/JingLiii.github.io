CSS背景,盒子模型的初级介绍

2016-06-18

背景

背景颜色,背景图片,以及进行背景图片的设置

  • background-image

  • none: 无背景图
  • url: 指定背景图的地址

    background-repeat

  • repeat: 平铺 默认值
  • no-repeat: 不平铺
  • repeat-x: 沿X轴方向进行平铺
  • repeat-y: 沿Y轴方向进行平铺

    background-position

  • length length,如果只是指定了一个值,则这个值表示的横坐标,第二个值表示的纵坐标
  • position position,top bottom center;left center right;都是表示位置,可以不分先后,一般情况下,将横坐标left写在第一个位置.

    background-attachment

    背景附着
  • scroll: 背景图像随对象内容进行滚动(默认)
  • fixed: 背景图像固定

    背景简写

    没有强制性的要求和规定

    盒子模型 重点掌握

    所有的文档元素(标签)都会生成一个矩形框,我们称为元素框,它描述了一个文档元素在网页布局中所占的位置大小.

    盒子边框

  • border: 宽度||样式||颜色
  • border-style: 上 右 下 左 : none(无,默认,也用于边框清零)
  • border-width: 顺序相同: 宽度为像素值
  • border-color: 边框颜色,可以采用十六进制的颜色
  • 综合设置的话,就是直接一个border: 四边的宽度,四边的样式,四边的颜色

    内边距

    padding: 用于设置边框和内容之间的距离
  • 分开设置: padding-top, padding-right, padding-bottm, padding-left.
  • 数值的个数不同,代表的含义不同
    • 1个值,代表各个方向的内边距
    • 2个值,代表上下内边距,左右内边距
    • 3个值,代表上内边距,左右内边距,下内边距
    • 4个值,代表上,右,下,左

      外边距

      margin: 元素之间的空白,通常情况下,这个空白不能再放置任何内容
  • 顺序与内边距相同
  • 水平居中
    • 块级元素
    • 指定宽度
    • 左右的外边距设置为auto

      清除内外边距

      通常情况下,使用通配符进行清除所有元素默认的内外边距,以免影响网页布局

行内元素只有左右内外边距,没有上下内外边距

外边距合并

  • 相邻块级元素,垂直外边距合并

  • 嵌套块级元素,垂直外边距河边

    盒子布局的稳定性

    宽度(width)>内边距(padding)>外边距(margin)
