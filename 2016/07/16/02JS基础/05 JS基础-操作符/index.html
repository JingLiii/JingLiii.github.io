JS基础-操作符

2016-07-16

数学运算符

  • “ + “: 加法运算符
    • 若是有一个操作符为字符串,就把另一个操作符转化为字符串,然后拼接起来
  • “ - “: 减法运算符
  • “ * “: 乘法运算符
  • “ / “: 除法运算符
    • “/0”: 对0做除法,返回Infinity
  • “ % “: 取余运算符
    • 非数值型取余，返回NAN

      一元运算符

      只能操作一个值的元素符称为一元运算符,分为前置型和后置型

      “ ++ “: 数值自增1;

      “ – “: 数值自减1;

      前置型: 数值在参与运算前先自增1,再参与运算

      后置型: 数值自增1,将变化后的数字保存起来,然后用变化前的数参与运算,当再次用到这个数值的时候,就采用变化后的数

      也就是第一次用到的时候并不会改变值,第二次用到的时候值就改变了

逻辑运算符 短路操作

&& 与运算 and

两个操作数同时为true，结果为true，否则都是false

  • 属于”短路”操作,即第一个值能够决定结果,就不会再对第二个值进行求值判断
  • 两个都为true,结果为true
  • 有一个结果为false,结果为false
  • 也就是说当第一个值为false时,就不会再运算第二个了,找false即可.

    || 或运算 or

    两个操作数有一个为true，结果为true，否则为false
  • 属于”短路”操作
  • 有一个为true,就是true
  • 两个都是false,结果是false
  • 也就是说,只要第一个为true,就不再进行后面的运算.

    ! 取反 not

    逻辑取反,true变false,false变true
  • 如果操作的不是布尔类型,会先对其进行转换,再对其取反

    短路操作

    && 与

    总结: 会从第一个操作数开始找false,找到就返回,未找到返回最后一个.
  • 如果第一个操作数是对象,就返回第二个操作数
  • 如果第一个操作数返回true,并且第二个操作时是对象,返回第二个操作数
  • 如果有一个数是null/NaN/undefiend,就返回null/NaN/undefiend
  • 操作是两边都不是布尔类型,会把操作数转换成布尔类型,返回值是false的那个操作数
  • 如果两边都是false,就会返回第一个.

    || 或

    总结: 会从第一个操作数开始找true,找到就返回,未找到返回最后一个.
  • 如果第一个操作数是对象,就返回第一个操作数
  • 如果第二个操作数是对象,并且第一个操作数是false,就返回第二个操作数
  • 两边都不是布尔类型的话,先转换,返回是true的那个操作数
  • 两边都是true,返回第一个

    比较运算符

    > < >= <=

  • 在比较两个字符串时,比较的是两个字符串对应位置的每个字符的字符编码值
  • 在比较数值和字符串时,字符串都会变转化成数值,然后以数值的方式与另一个数进行比较
  • 如果一边是数字,一边是Boolean,先把Boolean的值转换成数字在比较.
  • true转变为数字为1,false转变为数字为0

    == != === !==

  • 相等比较的是值,全等比较的是值和类型

赋值元素符

= += -= *= /= %=

运算符的优先级

从高到低:

  • (): 优先级最高
  • 一元运算符: ++ – !
  • 算数运算符: 先 * / % , 后 + -
  • 关系运算符: > >= < <=
  • 相等运算符: == != === !==
  • 逻辑运算符: 先 && , 后 ||
  • 赋值运算: = += -= *= /= %=
