switch

  • break语句可以跳出循环,若是没有break语句,会继续向下执行

  • switch(),括号中传入和值,在与下面的case进行判断对比的时候,判断的是’===’全等,也就是说需要判断数据类型.

  • 案例:转化分数

    var sco = window.prompt('请输入成绩');
if (sco != parseFloat(sco)) {
    console.log("输入有误");
} else if (sco > 100) {
    console.log("最大为100");
} else {
    sco = parseInt(sco / 10);
    switch (sco) {
        case 9:
            console.log(9);
            break;
        default:
            console.log(10)
    }
}
