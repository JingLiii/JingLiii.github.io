Number类型

数值类型

  • 其他进制都在进行运算时都将转换成十进制进行运算
  • 浮点数并不可靠,不要对浮点数进行判断.取整进行判断
  • 无穷大: Infinity; 无穷小: -Infinity;
  • NaN
    • Not a Number
    • NaN和任何数值都不相等,包括NaN本身
    • isNaN(): 任何不能被转换成数值的,都返回true;
      ##　String
      字符串
  • 使用单引号或者双引号包含
  • 由一个个字符组成,可通过.length属性获得长度
  • 字符串具有不可变性:
    • 一旦创建就不能改变
    • 如果栈当中的字符串重新赋值,会在栈中再次开辟一个空间,然后再把值删除,用新的值替换
    • 等到js中的垃圾回收机制进行回收
  • 字符串拼接:
    • 使用 + 进行字符串拼接
        • 是完全不同的
      • 可能会引起隐式类型转换
  • 转义字符:

    Boolean类型

  • 有两个变量true和false,区分大小写
  • 所有的数据类型都可以等价为boolean类型

  • Undefined类型

    特殊数据类型
  • 表示变量声明,而未赋值
  • 只有一个值undefind

两种类型的区别

基本数据类型传值, 引用数据类型传地址

存储方式

  • 基本数据类型的数据保存在栈中
  • 引用数据类型的数据保存在堆中, 栈中只是保存一个地址

    数据作函数的参数时的传递方式

  • 基本数据类型做参数时,传递的是数值
    • 会把x, y的值拷贝一份传递给a, b

  • 引用数据类型做参数时,传递的是地址
    • 会把数据的地址复制一份给参数
