JS基础-对象分类

2016-08-12

object对象:

js就是一门基于object的语言

内置对象

js内部已经定义好的对象: Array, String, Number, RegExp, Error

Array

  • 属性 length

String

  • 属性: length
  • 方法 indexof charAt

自定义对象

程序员使用构造函数定义的对象

封装函数

封装一个方法: 创建一个空白的对象,为他绑定属性和方法.再通过该函数进行对象的创建

构造函数

var obj = new Object()

  • new的作用
    1. 开辟一块内存空间. 把新建的对象放入其中
    2. 把this指向当前对象
    3. 把对象返回.
  • this的作用
    1. this只能出现在函数中.
    2. this代指函数的调用者.
    3. new Object(), this代指被创建出来的对象

DOM对象

网页上的标签元素都是DOM对象
